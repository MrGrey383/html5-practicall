My Journey in Technology

I'm currently pursuing my Bachelor's degree in Computer Science at the United States International University (USIU). My passion lies in creating accessible, user-friendly web experiences that make technology available to everyone.

My daily workflow involves coding with tools like VS Code, version control with Git, and testing applications using Ctrl + Shift + I for developer tools.

I'm particularly fascinated by semantic HTML elements like <main> and <section> that create meaningful document structure for both users and search engines.
